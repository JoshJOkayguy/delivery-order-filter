# Delivery Order Filter
### Disclaimer: This single page react app was made entirely by Claude in about 2 minutes. I'm using it as a personal tool, and it will remain public for anybody that would like to take advantage of it.

Simple application that allows you to set a base minimum rate/mile for delivery driving (Doordashing, UberEats, etc...). The app will then tell you based on your next order (input offered pay & miles) whether or not to take it based on your threshold.

Basically, this just saves you from doing basic math to determine if an order is worth taking or not.
